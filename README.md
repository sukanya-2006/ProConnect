
# 💼 ProConnect

ProConnect is a MERN-based professional networking platform inspired by LinkedIn, where users can create profiles, share posts with images, and connect with others through a professional social feed.

🚧 This project is currently under active development.

---

## Features Implemented

- User Authentication (Login & Register)
- Dashboard Feed
- Create Posts
- Upload Images with Posts
- Delete Posts
- Discover Users Section
- Redux Toolkit State Management
- Responsive Feed UI
- Dynamic Profile Rendering

---

## Tech Stack

### Frontend
- Next.js

## Setup Instructions

### Clone Repository

```bash
git clone https://github.com/sukanya-2006/ProConnect.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

### Backend Setup

```bash
cd backend
npm install
nodemon server.js
```

Backend runs on:

```bash
http://localhost:9090
```

# ProConnect

ProConnect is a full-stack professional networking platform inspired by LinkedIn, built using the MERN stack. It enables users to build professional profiles, connect with others, share updates, engage with posts, and showcase their professional journey in one place.

This project is currently under active development, with new features and improvements being added regularly.

---

# Live Demo

https://pro-connect-brown.vercel.app/

## Features

### Authentication & User Management

* User Registration
* User Login
* Secure Token-Based Authentication
* Dynamic User Profiles

### Professional Profiles

Users can create and customize their profiles with:

* Profile Picture Upload
* Banner Image Upload
* About Section
* Work Experience History
* Education History
* Skills Management
* Profile Completion Suggestions

### Social Feed

* Create Posts
* Share Text Posts
* Upload Images with Posts
* Delete Posts
* View Community Feed
* Responsive Feed Layout

### Post Engagement

* Like Posts
* Comment on Posts
* Dynamic Comment Rendering
* View Post Discussions

### Professional Networking

* Discover Users
* Send Connection Requests
* Accept Connection Requests
* Reject Connection Requests
* View Network Connections
* Connection Status Tracking

### Resume Generation

Generate professional resumes directly from profile information, including:

* Personal Information
* About Section
* Work Experience
* Education History
* Skills

### User Experience Improvements

* Responsive Design
* Mobile-Friendly Layouts
* Empty State Handling
* Dynamic Profile Rendering
* Fallback User Avatars
* Improved Error Handling
* Null-Safe Rendering
